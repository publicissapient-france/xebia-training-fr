Programme

Cette formation Kanban est une formation de terrain qui privilègie l'apprentissage à une formation purement théorique.

Cette formation est organisée sur deux jours. Elle comprend deux 1/2 journée de mise en pratique à la suite des éléments théoriques étudiés chaque matinée.

Première matinée :

  • Les bases de Kanban
    • Il était une fois Kanban
    • Kanban dans les méthodes Agile
    • Le Management visuel
  • Le management visuel Versus Kanban
  • Le système Push Versus Pull
  • Le mécanisme de flux continu
  • La limite sur le travail en cours
  • La cinématique Kanban
  • Les classes de service

Première après-midi :

  • Activité sur la mise en place d'un kanban :
    • serious game : joli tableau
    • Serious Game : Pizza Games

Deuxième matinée :

  • L'amélioration continu avec Kandan
  • Gérer les risques avec un Kanban
  • Comprendre et Analyser les indicateurs du Kanban
  • Réduction des délais et des couts avec Kanban
  • La chaine de valeur avec Kanban (Value stream Mapping)
  • La prédictibilité avec Kanban
  • Améliorer le processus avec le A3
  • Créer un Kanban de Backlog de produit
  • Créer un Kanban de projet
  • Le portfolio de projet
  • Ajuster une Kanban au fil du temps (PDCA)

Deuxième après-midi :

  • Activité de serious Game
  • Casino Game ou Beer Games
