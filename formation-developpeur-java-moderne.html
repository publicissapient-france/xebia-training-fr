Ben Evans

Ben Evans est le Directeur General de Teamsparq. Il évolue dans le monde de l’Open Source depuis les années 90. Il a piloté et délivré des projets d’envergure mondiale dans le secteur bancaire, des médias et des œuvres caritatives. Il évolue actuellement en tant qu’Architecte et Expert Java auprès d’une des plus grosses institutions financières mondiales.

Il a évolué notamment en tant que garant des performances d’une plate-forme boursière lors de l’introduction en bourse de Google, créateur des sites web de certains des plus grands films d’Hollywood ainsi que de certains des plus importants sites de e-Commerce du Royaume Uni et développeur de système d’échanges commerciaux à faible latence gérant des commandes avec des volumes dépassant les 1000 M de dollars par an. Il a aussi était impliqué dans les premiers essais du réseau 3G chez British Telecom.

Formation développeur Java moderne

PROGRAMME

Résumé

Cette session de 3 jours a pour objectif de raviver la passion et d’améliorer les compétences des développeurs. La formation apporte les bases essentielles de certaines méthodes parmi les plus utiles au développeur Java. La formation est un mélange de théorie et d’exercices pratiques sur des sujets tels que :

  • Pragmatic Software Development Life-Cycle (SDLC)
  • La conception et l’architecture pour le Test Driven Development (TDD)
  • Le Refactoring du code legacy
  • “Classloading, bytecode”
  • L’exploration du développement fonctionnel et les concepts JVM HotSpot
  • Une introduction à Java 7
  • L’adoption de « Java.util.concurrent »

Les questions abordées

  • Comment refondre une application afin d’adopter une approche orientée Test ?
  • Quels aspects de Hotspot JVM et platform sont les plus utiles aux développeurs d’applications ?
  • Quelles sont les nouveautés avec Java 7 ?
  • Java.util.concurrent VS Java “classic” concurrency, etc.
