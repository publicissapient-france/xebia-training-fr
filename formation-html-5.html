Séven est consultant chez Xebia, où il intervient en tant qu'architecte pour des sites web à fort trafic.

FORMATION HTML 5

présentation

Comment tirer partie des nouveauté HTML5 pour construire une application web riche disponible hors ligne. Cette formation alterne la présentation théorique des éléments et des API et la mise en pratique par des exercices sur un même projet clé en main qui vous permettra de connaître les best-practices et surtout de mettre en oeuvre les fonctionnalitées avancées d’HTML5. Nous tenterons de répondre à plusieurs questions par la pratique :

  • Comment construire le layout de son interface en optimisant son indexation pour les moteurs de recherche ?
  • Comment améliorer les performances JavaScript de l’interface web ?
  • Comment persister les données dans le navigateur ?
  • Comment rendre l’application disponible hors-ligne ?
  • Dans quels cas utiliser les WebSockets ?
  • Comment ajouter du contenu multimédia sans Flash ?
