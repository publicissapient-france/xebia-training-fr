Présentation

Téléchargez le programme PDF de la {{ page.titreofficiel }}

Cette formation de 2 jours est ponctuée d'échanges théoriques et d'ateliers Hands on. Vous aborderez notamment les sujets suivants : Apache Cassandra, Apache Hadoop, Apache Solr, DataStax Enterprise et DataStax OpsCenter.

Cette formation vous apportera tous les éléments dont vous avez besoin pour adopter avec succès Apache Cassandra. Les sujets abordés seront : les données NoSQL, les colonnes et les familles de colonnes, les comparateurs, L'API C*, l'indexation, l'indexation secondaire, les colonnes composites, les series ordonnées, les données temporelles, Hadoop, Hive, Pig et Solr.
