Introduction

Cette formation s’adresse à des débutants en Java, qui souhaitent consolider leurs connaissances du langage et de ses APIs les plus utilisées.

L’ensemble des concepts est présenté de façon précise avec de nombreux voyages dans le code source du JDK et dans la Javadoc, le but étant à la fois de démystifier ces pratiques, de les rendre naturelles, et surtout de montrer ce qu’elles peuvent apporter.

API de base (1 journée)

API Collection

  • Collection et de List : implémentations par tableaux et listes chaînées
  • Table de hachage : Map, SortedMap
  • Set, SortedSet : implémentation par table de hachage
  • Performances de tables de hachage, alternatives aux implémentations du JDK
  • Implémentation de caches par tables de hachage, cas des SoftReference
  • Concepts de file d’attente, interface Queue

API Java I/O

  • Notions de base : fichier, flux
  • Flux de caractères / flux binaires, et classes associées
  • Flux croisés
  • Serialization d’objet, principe et utilité
  • Écrire ses propres méthodes de sérialization
  • Flux compressés

API Java NIO 2

  • Notions de Path
  • Classe Files
  • Notion de FileSystem, de FileStore
  • Gestion du système de fichiers
  • Arborescence de répertoire
  • Observation d’un répertoire
