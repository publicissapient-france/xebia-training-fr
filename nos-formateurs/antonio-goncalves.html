Formateur Java Antonio Goncalves

Antonio Goncalves (Champion Java, leader du JUG)

Membre du Java Community Process sur Java EE 6, Antonio est avant tout un homme de terrain qui conseille ses clients et développe leurs architectures. Co-leader du Paris Java User Group, animateur du podcast Les CastCodeurs et Java Champions, il partagera ici son expérience et sa passion des architectures. Antonio est l’auteur des livres Java EE 5 et Java EE 6.
