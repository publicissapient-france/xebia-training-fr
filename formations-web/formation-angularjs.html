Présentation Formation AngularJS Framework

Objectifs de la formation AngularJS

  • Vous faire comprendre AngularJS et son intérêt pour la création de site web moderne et réactif
  • Vous apprendre les bases du framework pour être efficace rapidement tout en ayant de bonnes pratiques
  • Vous permettre de bâtir votre première application en mode "test first" qui communiquera avec une API REST

Cette formation dure deux jours et est basé sur la dernière version du framework.

Vous avez déjà participé à cette formation ? Recommandez là sur LinkedIn !

Programme de la Formation AngularJS Framework

1ère journée : les bases de AngularJS

Le but de cette première journée est de découvrir Angular et de créer sa première application

  • La philosophie d'Angular
    • Découvrez les concepts généraux du framework pour bien débuter et la mise en place d'un premier projet.
  • Single Page Application
  • MVC côté client
  • Un DSL en HTML
  • Les outils et les tests
  • Les bases
    • Créez votre première application en découvrant les bases du framework
  • La vue et les directives
  • Le modèle et le Scope
  • Le controller
  • Communiquer avec le serveur
  • Le routeur
  • Les filtres

2ème journée : AngularJS en pratique

Le but de cette seconde journée est de pousser les concepts du Framework et de mettre en place certaines bonnes pratiques

  • Les directives de tous les jours
  • Créer ses propres filtres
  • Créer ses propres services
  • Créer ses propres directives
  • Les modules
  • Les intercepteurs
  • Intégration dans votre projet
