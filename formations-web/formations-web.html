Formation Ember.JS/ Mathieu Breton et Benoit Lemoine

Le but de la formation est d'apprendre à des personnes ayant un minimum de connaissances en JavaScript :

  • les bases du framework MVC Ember.JS
  • à bâtir une application cliente (dans un navigateur) qui communiquera avec une API REST

Cette formation dure trois jours et est basée sur la dernière version en date de framework MVC Ember.JS, à savoir la version 1.0RC1.

  • Durée : 3 jours
