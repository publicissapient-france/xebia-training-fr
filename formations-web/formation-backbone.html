Programme

Enjeux

  • S'approprier Backbone, Underscore et jQuery
  • Structurer une application en composants modulaires
  • Tester une application en unitaire et en intégration
  • Générer un livrable prêt pour la production

Mise en route

  • La single page application
  • Le MVC selon Backbone
  • L'intégration élégante de jQuery et d'Underscore dans Backbone
  • Les tests unitaires et d'intégration

Prise en main

  • Vue + Template : créer un composant
  • Modèle + Collection : requêter un serveur
  • Routeur : gérer la navigation par url et hashtag

Utilisation avancée

  • Éditer, filtrer et trier une collection
  • Composer une vue de sous vues
  • Choisir un moteur de templating
  • Intégrer du drag&drop avec jQuery UI
  • Découper une application en modules
  • Optimiser une application (sous-vues, vues fantômes, etc.)
  • S'outiller avec Grunt, de l'intégration continue, choisir un IDE, utiliser un preprocessor css, etc.

Bio

Yves Amsellem

Développeur depuis 6 ans — les 3 derniers chez Xebia — Yves tire de son expérience sur des sites à fort trafic une culture de la qualité, de l'effort commun et de l'innovation. Il intervient depuis 2 ans sur des projets web Backbone reposant sur ReST et NoSQL. Avec Benoît Guérout, il développe la librairie open source Jongo — Query in Java as in Mongo shell.
